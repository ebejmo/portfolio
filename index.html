Three put blog...

The task for this project was to create a blog site, design and topics on the blog was chosen by myself. Content is fetched from a headless WordPress CMS and pages are dynamically created based on what link the user has clicked.

Is the content on the site amazing? Absolutely not. But the site is looks good on diffrent screen sizes and is inutivie and user friendly. On the homepage there is a sweet carousel that displays all the blogposts and on the blog page there is a "Show More" button that does what it supposed to do. Since I am just getting strated with learning JavaScript this project was by far the most challenging to date but also where I have learned the most.
