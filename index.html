Three put blog...

The task for this project was to create a blog site, design and topics on the blog was chosen by myself. Content is fetched from a headless WordPress CMS and pages are dynamically created based on the users chosen link.

Is the content on the site exceptional? Absolutely not. However, the website offers a visually appealing design that adapts seamlessly to different screen sizes, ensuring an intuitive and user-friendly experience. The homepage features a carousel that showcases all the blog posts, while the blog page incorporates a functional 'Show More' button. As a beginner in JavaScript, this proved to be the most challenging project to date, yet providing valuable lessons and important knowledge.

rainydays

This project started with designing a prototype using Adobe XD, envisioning an online rain jacket store for a clothing company. The 'Rainydays' project evolved into a fully functional live website, crafted using fundamental HTML and CSS techniques. Leveraging the power of JavaScript, dynamic page creation was implemented, while content management and product retrieval were seamlessly integrated using the WordPress REST API. While the site is currently undergoing construction, it is worth noting that the 'add to cart' functionality is being developed to enhance the overall user experience.
